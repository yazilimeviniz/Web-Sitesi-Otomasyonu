### 🇹🇷 Türkçe Proje Tanıtımı

**Yazılım Eviniz Otonom**, şirket bilgilerine, sektöre ve seçilen parametrelere göre üst düzey web sitelerini, dönüşüm oranı optimize edilmiş metinleri, sektöre özel medya kaynaklarını ve üst düzey teknik SEO şemalarını saniyeler içerisinde üreten **Yapay Zeka Destekli Otonom bir Dijital Ajans Üretim Bandıdır**.

"İnsan vizyonu kurar, yapay zeka tuğlaları dizer" felsefesiyle tasarlanan bu platform, geleneksel kodlama yükünü sıfıra indirir. AI destekli kararsızlık çözümleyicileri sayesinde boş veya belirsiz bırakılan tüm teknik tercihler ve görsel şemalar, yapay zeka tarafından otonom olarak doldurulur.

#### 🌟 Öne Çıkan Özellikler

1. **Gelişmiş Yapay Zeka Karar Mekanizması**: `@google/genai` TypeScript SDK'sı ile güçlendirilmiş Gemini Pro-Intel motoru, firmanızı analiz eder ve en yüksek dönüşüm getiren içerik, servis ve pazarlama kurgularını üretir.
2. **Zengin Görsel Stil & Tema Kütüphanesi**:
   - **Tasarımcı Neon Gece (Neon Noir)**: Koyu karbon tonları, siber mor ve parlak neon akımları.
   - **Sadelik Japandi (Japandi)**: Sıcak ahşap tınıları, doğal bej rengi ve siber zen dinginliği.
   - **Organik Eko-Doku (Organic Eco)**: Doğa dostu yeşil tonlar, taze organik yaprak kremi zeminler.
   - **Geometrik Retro Bauhaus (Bauhaus)**: Mat primer renkler, cesur geometrik kalın çerçeveler.
   - **Monokrom Kod Terminali (Cyber Terminal)**: Derin siber-siyah zemin üzerinde parıldayan yeşil kod matrisi.
   - *Lüks Butik (Luxury Editorial), Apple Minimalizmi (Apple Clean) ve Brutalist Sokak Retro temaları.*
3. **Yaratıcı Hero (Giriş) Tasarımları**:
   - **Interactive Image Slider**: Otomatik geçiş ve animasyonlu slayt kontrollerine sahip görsel karoseli.
   - **Full Video / Cinematic Background**: Arka planda 60 FPS otonom video siber-dokusu simüle eden sinematik sunum.
   - *Split-Image, Glassmorphism Glow, Futuristic Dashboard ve Apple Minimalist yerleşimleri.*
4. **Hava Geçiren Asimetrik Galeri Varyasyonları**:
   - **Justified Bento Box**: Göze çarpan asimetrik geometrik bento yerleşimi.
   - **Stacked Premium Mosaic**: Modern sıkıştırılmış gölgesiz mozaik kartları.
   - **Cyber Darkroom (Karanlık Stüdyo)**: Analog film karesi görünümlü siyah-beyaz retro arşiv galerisi.
   - **Creative Neon Glow**: Aura etkili, parlayan neon mor çerçeveli yaratıcı galeri.
5. **Akıllı "AI Seçsin" Algoritması**: Seçim yapmadığınız detayları otonom yapay zeka hedeflerine göre otomatik yapılandırır.
6. **Yüksek Performans & SEO Optimizasyonu**: Otomatik oluşturulan schema.org mikro-verileri, WebP sıkıştırılmış hazır Unsplash medya kütüphanesi ve anlık SEO denetim aracı.
7. **Yerel Veri Depolama & Çevrimdışı Portfolyo**: Üretilen tüm web siteleri tarayıcınızın `localStorage` önbelleğine kaydedilir, böylece otonom üretimleri dilediğiniz zaman inceleyip yönetebilirsiniz.
8. **Dışa Aktarma Merkezi (Export Hub)**: Oluşturduğunuz otonom web sitesini tek bir tıklama ile temiz ve sadeleştirilmiş bir biçimde dışarı aktarın.

### 🇺🇸 English Project Overview

**Yazılım Eviniz Otonom** is a **State-of-the-Art Autonomous AI-Powered Web Production Pipeline** that constructs fully functional digital agency websites, conversion-optimized content copies, curated stock media, and technical SEO structure schemas in seconds.

Built upon the philosophy of *"Humans direct the vision, AI places the bricks,"* the platform completely eliminates manual grunt work. Whenever details are left undecided, the autonomous design neural model dynamically populates color theory, UX visual frameworks, layouts, and copy configurations.

#### 🌟 Key Features

1. **Incredibly Intelligent AI Core**: Driven by the `@google/genai` SDK and Gemini Pro-Intel model architectures to analyze target audience metrics and deliver the highest possible conversion rate optimization (CRO).
2. **Distinctive Visual Aesthetic Themes**:
   - **Designer Neon Noir**: Deep carbon canvas, glowing cyber-violet gradients, and lush pink neon highlights.
   - **Minimalist Japandi**: Warm sand hues, natural wood textures, and comforting asymmetrical Zen space.
   - **Organic Eco-Texture**: Forest greens, natural leaf cream papers, and environmental visual anchors.
   - **Geometric Retro Bauhaus**: Flat primary colors (red, blue, mustard yellow), bold grids, and thick lines.
   - **Monochrome Code Terminal**: Dark terminal black canvas accented by glowing green telemetry data streams.
   - *Luxury Boutique (Editorial), Apple Fluid Minimalist, and Brutalist Street frames.*
3. **Advanced Hero Slider Layouts**:
   - **Interactive Image Slider**: Gorgeous responsive image carousel utilizing automatic timers and fade animations.
   - **Full Video / Cinematic Slider**: Cinematic presentation with simulated movie backgrounds, telemetry grids, and video telemetry dashboards.
   - *Split-Image Grid, Elegant Centered, Glassmorphic Glow, and Apple aesthetics.*
4. **Asymmetrical Fluid Gallery Grids**:
   - **Justified Bento Boxes**: Elegant modern asymmetrical layout structures.
   - **Stacked Mosaic**: Tight, gapless premium mosaic photo cells.
   - **Cyber Darkroom**: Contrast-heavy grayscale cinematic archival cells resembling physical film frame reels.
   - **Creative Neon Glow**: Dark glow aesthetics framing visual assets with soft cosmic outline blooms.
5. **Smart "AI-Choose" Algorithms**: Autonomously evaluates company goals and targets if users choose not to detail specifications.
6. **Optimized Technical SEO Schemas**: Instant valid schema.org generation, structured micro-rich annotations, and on-page visual performance audits.
7. **Local Portfolio Offline Storage**: Automatically saves all generated clients locally using persistent client-side `localStorage` stores.
8. **Export Hub**: Download and transfer finalized aesthetic responsive single-page static designs instantly.

## 🛠️ Kurulum ve Çalıştırma / Installation and Getting Started

### Gereksinimler / Prerequisites
* **Node.js**: v18+ (v20 veya üzeri tavsiye edilir)
* **NPM** / Yarn / Pnpm

### 1. Depoyu Klonlayın ve Bağımlılıkları Yükleyin / Clone the Repository & Install Dependencies
```bash
git clone <repository-url>
cd <repository-directory>
npm install
```

### 2. Çevre Değişkenlerini Tanımlayın / Set Up Environment Variables
Kök dizindeki `.env.example` dosyasını kopyalayarak `.env` adında yeni bir dosya oluşturun ve Google Gemini API anahtarınızı ekleyin. (API Anahtarınız olmadığında sistem otonom yerel şablon algoritmasına otomatik olarak geçer).

Copy `.env.example` to `.env` and configure your Google Gemini API Key. (If left empty, the engine automatically defaults to local fallback code structures seamlessly).

```env
GEMINI_API_KEY=your_gemini_api_key_here
```

### 3. Geliştirme Sunucusunu Başlatın / Run Development Server
Hem Express backend API katmanını hem de Vite istemci arayüzünü tek seferde başlatmak için:

To start both the Express backend API proxy and the frontend Vite server concurrently in development mode:
```bash
npm run dev
```
Uygulamanız varsayılan olarak `http://localhost:3000` adresinde aktif olacaktır. / Your applet will start running on port `3000`.

## 🚀 Üretim Derlemesi ve Dağıtım / Production Build and Deployment

Uygulama tam uyumlu bir bulut konteyner mimarisine sahiptir ve Cloud Run veya herhangi bir Node.js sunucusunda çalışmaya hazırdır.

The application has a fully integrated cloud-native container architecture, prepared to run in Cloud Run or any standard Node.js VM.

### 1. Derleme (Build)
İstemci tarafını optimize etmek ve Express backend sunucusunu tek bir CommonJS dosyasına paketlemek amacıyla esbuild betiğini çalıştırır:

Generates optimized frontend assets and bundles the backend server into a single self-contained `dist/server.cjs` via `esbuild`:
```bash
npm run build
```

### 2. Üretim Sunucusunu Başlatma (Start)
Derlenen optimize edilmiş üretim paketini çalıştırmak için:

To boot up the compiled production deployment server:
```bash
npm run start
```

## 🎨 Tasarım ve Tipografi Seçimleri / Design and Typography Philosophy

Bu uygulama, estetik lüksü ve dengeli boşlukları (generous negative space) her şeyin önünde tutar:
* **Yazı Tipleri (Typography)**: Display başlıklarında fütüristik **Space Grotesk** ile **Outfit**; gövde metinlerinde ise göz yormayan, okunaklılığı yüksek **Inter** ve kod bloklarında hassas **Fira Code** bir araya getirilmiştir.
* **Modern Kartlar**: Klasik gri kutular yerine, siber neon auralar içeren cam buzlama katmanları (glassmorphic glow) ve kenarları keskin brutalist çerçeve yapıları tercih edilmiştir.
* **Gürültüsüz Arayüz (Anti-AI-Slop)**: Marginlerde ve alt başlıklarda sahte telemetri logları ("STATUS: ONLINE", "PORT: 3000") gibi gereksiz teknik kalabalıklar yerine son derece sade, işlevsel ve asil bir kurumsal tasarım dili oturtulmuştur.
